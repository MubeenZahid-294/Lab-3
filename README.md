# 🔐 PHP Authentication System

A simple **Authentication System** built with **PHP** and **MySQL**, including user signup, login, logout, and a protected dashboard.

---

## ✨ Features
- 📝 User Signup Page with validation  
- 🔑 Login System with session management  
- 🚪 Logout functionality  
- 📊 Dashboard page accessible only after login  
- ✅ Secure password hashing
